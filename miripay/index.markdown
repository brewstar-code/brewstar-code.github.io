## 문제 인식: 선불권은 아직도 왜 수기일까요?

### 선불권이 수기인 이유
- **가맹점에 유리하기 때문입니다**
- **현금 흐름이 좋기 때문입니다**
- **복잡한 결제 시스템이 필요 없기 때문입니다**

### 하지만 문제는 관리입니다
- **종이 장부**
- **엑셀**
- **기억에 의존한 잔액**

👉 돈은 받았는데, **남은 금액을 헷갈리는 순간**<br>
👉 고객도, 사장님도 불편해집니다.

---

## 미리페이의 해결 방식

### 수기는 그대로 두세요
### 관리만 디지털로 바꿉니다

#### 📲 QR 코드 스캔
- 고객 선불권 QR 생성
- 스캔 한 번으로 고객 선택

#### 💰 금액 입력
- "얼마 받았어요?"
- 입력 끝

#### 🧾 자동 기록
- 언제
- 얼마를
- 누가
  → 전부 자동 저장

## 미리페이가 특별한 이유

### ✔ 기존 선불권 유지
- 고객도 익숙
- 사장님도 편함

### ✔ 도입 장벽 0
- 카드기 교체 없음
- 계약 없음
- 복잡한 설정 없음

### ✔ 공유 가능한 선불권
- 가족
- 연인
- 직원
  → 하나의 선불권을 함께 사용

### ✔ 분쟁 감소
- "얼마 남았죠?"
- 기록으로 바로 확인
